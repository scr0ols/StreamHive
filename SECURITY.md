# Security Policy

StreamHive is currently in early development and not yet deployed to
production. There are no versioned releases yet, so there's nothing to list
in a supported-versions table — this policy will gain one once the project
ships its first stable release.

## Reporting a Vulnerability

If you find a security vulnerability, please report it privately using
GitHub's [Private vulnerability reporting](../../security/advisories/new)
feature rather than opening a public issue.

As a solo project, response times aren't guaranteed, but reports will be
acknowledged and looked at as soon as possible.
